# PulseAnalytics

A pure Java-based trend analytics engine that performs:

- Frequency Analysis
- Grouped Frequency Mapping
- Top Trend Detection
- Data Aggregation

Built using:
- HashMap
- ArrayList
- TreeMap
- Java Collections Framework

## Upcoming Features

- File-based analytics
- Top-K trending system
- Real-time stream processing
- Report exporting
- GUI dashboard
- Spring Boot APIs
